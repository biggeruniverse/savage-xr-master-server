    Savage XR Master Server

    The repository hosts the code of the Savage XR Master Server. The whole responsibility of the master server is it to generate a server list for Savage XR clients. To do that the master server stores state information provided by Savage XR gaming servers via heart-beating.

    Requirements

    Java 1.7

    Getting Started

    Simply download the savage-master-server-1.0.5.jar

    Open a command shell and start the server with the following command:

    • java -jar savage-master-server-1.0.5.jar

    Configuration

    The server allows to configure some properties, which are listed below:

    listening port of the master server, default: 11236

    • listening.port = 11236

    path and name of the dat file to be written, default: ./gamelist_full.dat
